# Traffic Light Simulator in Java (GUI)

This is a simple Traffic Light Simulator built using Java Swing.

## Features
- GUI-based application
- Uses radio buttons to select traffic lights
- Displays message:
  - Red → Stop
  - Yellow → Ready
  - Green → Go
- Changes text color dynamically

## Technologies Used
- Java
- Swing (JFrame, JLabel, JRadioButton)
- AWT (Color, Font)
- Event Handling (ActionListener)
